# 教学激励系统

一款面向教育场景的前端教学激励软件，支持打包为桌面应用程序。

## 功能特性

### 学生端
- 🎒 **角色系统**：自定义角色外观，等级进化机制
- 🐾 **宠物养成**：领养宠物、装备系统、技能学习
- ⚔️ **PK对战**：与其他同学的宠物进行对战
- 📋 **每日任务**：完成学习任务获取奖励
- 🏆 **排行榜**：积分、等级、宠物等多维度排名
- 🎖️ **成就系统**：解锁成就获得特殊奖励
- 💬 **社交互动**：查看同学宠物，发起挑战

### 教师端
- 👥 **学生管理**：查看所有学生信息
- 🏆 **奖惩系统**：对学生进行积分、金币、经验的奖励或处罚
- 🐾 **宠物素材管理**：上传宠物GIF素材，设置进化等级和条件

## 技术栈

- **前端框架**: Vue 3 + TypeScript
- **状态管理**: Pinia
- **路由**: Vue Router
- **桌面打包**: Electron
- **构建工具**: Vite
- **样式**: CSS Variables + Animate.css

## 核心系统说明

### 积分奖惩机制
- 教师可对学生的作业完成、课堂表现、出勤等进行奖励
- 支持扣除积分、金币、经验值
- 经验值低于阈值时触发角色降级

### 宠物系统
- 5种宠物类型：小火龙、凤凰雏鸟、小老虎、萌萌兔、玄龟
- 5个进化等级：基础 → 进阶 → 中级 → 大师 → 传说
- 装备系统：武器、护甲、饰品
- 技能学习：根据等级和装备解锁技能
- PK对战：自动战斗系统
- 进化机制：满足等级、经验、胜利场次、任务完成数条件后自动/手动进化
- 形态切换：流畅的进化动画效果，记录进化历史

### 等级系统
- 10个等级，每级需要累积经验值
- 每5级触发角色外观进化
- 等级称号：新手学徒 → 传奇学霸
